# Chaiwala Webpage

## Overview
The Chaiwala webpage is a fun and interactive web application designed to provide users with a unique experience centered around chai (tea) and academic vibes. It features a playful design, engaging animations, and a GPA calculator to help students manage their academic performance.

## Project Structure
The project consists of the following files:
- `index.html`: Contains the HTML structure for the webpage.
- `styles.css`: Defines the visual appearance of the webpage.
- `script.js`: Handles interactivity and dynamic content updates.
- `README.md`: Provides documentation for the project.
- `LICENSE`: Contains the MIT License text.

## Setup Instructions
1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd chaiwala-webpage
   ```
3. Open `index.html` in your web browser to view the webpage.

## Usage Guidelines
- The webpage features background music that plays automatically. Users can toggle the music on and off using the provided buttons.
- Click on the interactive chai cup to increase your "Aura" score.
- Explore the special menu items and the GPA calculator for a fun experience.
